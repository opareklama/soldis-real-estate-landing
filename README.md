## Overview

SOLDIS Landing is a custom-built, modular architecture plugin that powers the SOLDIS Real Estate landing page. It replaces the need for heavy page builders (like Elementor) by providing a lightweight, ultra-fast, and highly secure native WordPress experience.

The plugin features a **SaaS-grade Command Center** dashboard and independent, modular settings panels for each section of the landing page, allowing the client to easily manage content without breaking the design.

## Features

- ⚡ **Zero-Dependency Frontend**: 100% Vanilla JavaScript and custom CSS. No jQuery, Bootstrap, or heavy libraries loaded on the frontend.
- 🏗️ **Modular Architecture**: Header, Footer, Hero, Benefits, Process, Investment, and FAQ are all independent modules.
- 🎨 **Premium UI/UX**: Designed with luxury real estate aesthetics, featuring glassmorphism, subtle micro-animations, and gold accents.
- 🛡️ **Enterprise Security**: Strict nonce verification, sanitization, and WP capabilities checking.
- 🚀 **Performance Optimized**: Lazy loading, intelligent asset enqueueing, and zero layout shifts.
- 📊 **Command Center**: A read-only health dashboard providing live checks on SEO, AEO, GEO, and Technical performance metrics.

## Engineering Philosophy

This plugin strictly adheres to the engineering standards of **OPA Reklama**:
- **Clean Architecture:** No hardcoded content. Fully database-driven via the WP Options API.
- **Maintainability:** Logical separation of concerns (Admin, Renderer, Views).
- **Client Experience:** A clutter-free, premium WordPress admin dashboard.

## Installation

1. Clone or download the repository into your WordPress `wp-content/plugins/` directory:
   ```bash
   git clone https://github.com/opareklama/soldis-real-estate-landing.git soldis-landing
   ```
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **SOLDIS Landing** in the WordPress admin sidebar to configure the Global Settings and Homepage sections.
