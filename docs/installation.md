# Installation Guide

## Prerequisites
- A running WordPress 6.0+ instance.
- PHP 8.1+.

## Setup Steps
1. Clone the repository into `wp-content/plugins/soldis-landing`.
2. Activate the plugin via the WordPress Admin Dashboard or WP-CLI (`wp plugin activate soldis-landing`).
3. Navigate to the **SOLDIS Landing** menu to access the Command Center.
4. Configure the Global Settings (Logos, Colors, Typography).
