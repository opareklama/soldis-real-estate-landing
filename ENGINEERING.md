# Engineering Philosophy

Developed by OPA Reklama, this project adheres to strict, long-term engineering principles:

1. **Zero Bloat:** Only load what is necessary. No generic third-party frameworks.
2. **Security First:** Never trust user input. Escaping, sanitization, and nonces are mandatory at all boundaries.
3. **Data Ownership:** No hardcoded client data. Everything is database-driven and editable via the WP Admin interface.
4. **Performance:** Sub-second rendering times through conditional asset loading and optimized DOM structures.
