# Support Guidelines

Thank you for using the **SOLDIS Landing** engine.

Because this is a bespoke, proprietary application built specifically for the SOLDIS Real Estate project, public support channels (like GitHub Issues) are not actively monitored for general inquiries.

## How to Get Support

For all technical assistance, feature requests, and bug reports, you must contact the OPA Reklama engineering team directly.

👉 **[Contact OPA Reklama](https://www.opareklama.lt/susisiekite-su-mumis/)**

### What to Include in Your Request

To help us resolve your issue as quickly as possible, please provide:
1. A clear description of the issue.
2. The current version of the plugin (you can find this in the Command Center Dashboard).
3. The WordPress version and PHP version of your server.
4. Screenshots or screen recordings demonstrating the problem.

Our team will review your request and deploy an Over-The-Air (OTA) update if a technical patch is required.
