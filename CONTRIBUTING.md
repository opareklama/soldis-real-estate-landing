# Contributing to SOLDIS Landing

This is a proprietary plugin engineered by OPA Reklama. While the repository is public, it is not an open-source project open for community contributions.

Internal OPA Reklama engineers should follow the guidelines outlined in the [Coding Standards](docs/coding-standards.md) document when pushing updates.

## Pull Requests
All pull requests must pass the internal CI/CD pipeline, including PHPCS checks against WordPress coding standards, before being merged into `main`.
